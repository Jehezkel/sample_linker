# Sample linker

## Allegro helper

Sample project to practice nodejs/react

1. Edit database.env - fill database password. <br/>
2. Start postgres on docker:

```
docker compose up -d
```

### Backend

- Prisma - orm for connection with database and models
- Jest - for testing.
- Express - for coffe
